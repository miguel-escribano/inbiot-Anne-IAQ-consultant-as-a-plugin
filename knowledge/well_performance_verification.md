## The 3 Continuous Monitoring Strategy Types

> Source: https://support.wellcertified.com/hc/en-us/articles/25695345374999

WELL v2 has three distinct strategy types for continuous monitors. Each has different verification requirements.

### 1. Monitor Deployment (informative — no threshold compliance required)

**Features:** A08 Part 1 (Install Indoor Air Monitors), T06 Part 1 (Monitor Thermal Environment)

**What it means:** The sensor just needs to be present, installed correctly, and maintained. No data needs to prove compliance with a threshold.

**Verification documents:**
- Engineer's Letter of Assurance + on-site photographs (at initial certification; again if changes at recertification)
- On-going Maintenance Reports (annually, after certification — proof of calibration per PVG frequency)
- **Sensor measurement data is NOT required** (as of WELL Addenda Q1–Q2 2024 and newer)
- Thermal comfort (T06): calibration requirement removed for ongoing verification

> Prior to Q4 2023 addenda: annual On-going Data Reports with measurement data were required. Projects on older addenda should verify which applies.

---

### 2. Performance-Based (threshold compliance via Sensor Data)

**Features that require sensor data proving compliance:**

| Feature | What to measure |
|---------|----------------|
| A01 Part 1 | PM2.5 / PM10 (Options 1/2/3) |
| A01 Part 2 | TVOC (Option 2: continuous monitoring) |
| A01 Part 3 | Inorganic gases (CO, O3, NO2) |
| A01 Part 4 | Radon (Option 1) |
| A03 Part 1 | CO2 — ventilation monitoring (Option 4) |
| A05 Part 1 | Enhanced PM thresholds |
| A05 Part 3 | Enhanced inorganic gases |
| A06 Part 1 | CO2 — enhanced ventilation (Option 4) |
| T01 Part 1 | Temperature + RH — long-term thermal data (Option 2) |
| T07 Part 1 | RH — long-term humidity data (Option 3) |

**Verification: Sensor Data submission**
- Minimum **1 calendar month** of data for initial certification/award
- For recertification/renewal: data from the entire period since previous award date
- Must be submitted using the WELL Sensor Data Template (updated January 29, 2026; requires Office 365)
- Template calculates minimum sensor quantity from project area (per PVG Table 2)

**Required submission package:**
1. Monitor Master List (all monitors + static parameters)
2. Sensor data file
3. Annotated floor plan (project size + monitor locations)
4. Photographs of each monitor (showing height, context, away from influences)
5. Documentation for ceiling placement if applicable

**Compliance rule:** ≥90% of sensor readings during occupied hours must meet the threshold. Missing data counts as exceedance.

---

### 3. Periodic Measurement (informative — no threshold compliance)

**Features:** A01 Part 5 (Measure Air Parameters), T01 Part 2 Option 1 (Semi-annual testing)

**What it means:** Parameters are measured periodically and uploaded as On-going Data Reports. No threshold compliance required — these are awareness/informative measurements.

**Verification:** On-going Data Reports uploaded annually to WELL Online after certification.
- Measurement frequency per feature requirements (e.g., twice per year for thermal)
- Does NOT need to follow device specs from the PVG (feature note applies)

## CONTINUOUS MONITORING REQUIREMENTS

### Densidad de Monitores

| Área Ocupable | Densidad | Mínimo |
|---------------|----------|--------|
| < 3,250 m² | 1 monitor / 325 m² | 2 |
| 3,250 - 25,000 m² | 1 monitor / 500 m² | 10 |
| > 25,000 m² | 1 monitor / 1,000 m² | 50 |
| Radon | 1 monitor / 2,300 m² | - |

### Ubicación de Monitores

- **Pared**: Altura 1.1 - 1.7 m sobre el suelo
- **Techo**: Solo si altura ≤ 3.7 m y aire mezclado uniformemente
- **Distancia mínima**: 1 m de puertas interiores, ventanas, humidificadores, impresoras
- **Evitar**: Flujo directo de HVAC o purificadores

### Especificaciones Técnicas de Sensores

| Parámetro | Tipo Sensor | Rango | Precisión | Resolución |
|-----------|-------------|-------|-----------|------------|
| PM2.5/PM10 | Óptico/Láser | 1-1000 μg/m³ | ±5 μg/m³ + 20% | 1 μg/m³ |
| TVOC | Electroquímico/MOS | 10-2000 μg/m³ | ±20 μg/m³ + 15% | 10 μg/m³ |
| CO | Electroquímico/MOS | 0.1-25 ppm | ±1 ppm (<10 ppm) | 0.1 ppm |
| O3 | Electroquímico | 10-500 ppb | ±10 ppb (0-100) | 5 ppb |
| CO2 | NDIR | 400-5000 ppm | ±50 ppm + 5% | 1 ppm |
| NO2 | Electroquímico/MOS | 5-500 ppb | ±20 ppb (0-100) | 1 ppb |
| Temperatura | Cualquiera | 10-40°C | ±0.5°C | 0.5°C |
| Humedad | Cualquiera | 5-95% | ±5% (10-90%) | 1% |
| Radon | Alpha track | 0.1-500 pCi/L | ±5% | 0.1 pCi/L |
| HCHO | Electroquímico/MOS | 20-1000 ppb | ±20 ppb (0-100) | 1 ppb |

### Intervalos de Medición

- **Radon**: ≤ 1 hora
- **Otros parámetros**: ≤ 15 minutos

### Calibración

- Recalibrar o reemplazar sensores cada **3 años**
- Opciones:
  - Calibración del fabricante
  - Calibración de campo con sensor de referencia (mínimo 2 concentraciones)

### Cálculo de Compliance (Continuous Monitoring)

**Regla del 90%**: Al menos 90% de los datos de cada sensor durante horas ocupadas deben cumplir los umbrales.

Ejemplo: Ocupación 08:00-18:00 (10h), intervalos de 15 min, 30 días:
- 10 h/día × 4 mediciones/h × 30 días × 90% = **1,080 mediciones** deben cumplir

## WELL CORE GUIDANCE

Para proyectos WELL Core (solo A01, no A05):
- Calcular muestras usando área no arrendada O 2.5% del área total (lo que sea mayor)
- Si área no arrendada ≥ 2.5%: todas las muestras en espacio no arrendado
- Si área no arrendada < 2.5%: al menos 1 muestra en cada tipo de espacio

Para proyectos WELL Core (A01 + A05):
- Calcular total de muestras con área total del proyecto
- Calcular mínimo en espacio no arrendado con área no arrendada
- Resto de muestras en espacio arrendado

## NOTAS IMPORTANTES PARA ANNE

1. **Sin tolerancia en monitoreo continuo**: Las tolerancias solo aplican a Performance Testing, NO a datos de sensores continuos.

2. **Datos faltantes = excedencia**: En monitoreo continuo, datos perdidos se interpretan como excedencia del umbral.

3. **RESET Grade B+**: Monitores acreditados RESET Grade B o superior son aceptables para WELL.

4. **InBiot MICA WELL**: Los sensores InBiot cumplen con las especificaciones técnicas de la Tabla 3 del PVG.

5. **Unidades**: 
   - TVOC debe reportarse en μg/m³ para A01.2
   - 1 ppb TVOC = 4.57 μg/m³ (calibración etanol)
