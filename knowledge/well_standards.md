## Certification Scoring

WELL certification is **points-based**, not threshold-based. All preconditions must pass regardless of target level.

| Level | Points required | Notes |
|-------|----------------|-------|
| Silver | 50 | All preconditions must pass |
| Gold | 60 | All preconditions must pass |
| Platinum | 80 | All preconditions must pass |

Total available: 100 points (92 optimizations across all concepts). Max 12 points per concept from optimizations.

**Preconditions** = mandatory for any certification level. Failing any single precondition = no certification.
**Optimizations** = optional; accumulate points toward Silver/Gold/Platinum.

WELL v2 has 23 preconditions total. Air concept has 4 preconditions (A01, A02, A03, A04).

## Air Concept: Features A01–A08

| Feature | Type | What it requires | Sensor-verifiable with MICA |
|---------|------|-----------------|----------------------------|
| **A01 Air Quality** | Precondition | PM2.5, PM10, TVOC, CO, O3, radon thresholds (5 parts) | Parts 1–3 yes; Part 4 (radon) no; Part 5 periodic |
| **A02 Smoke-Free Environment** | Precondition | No smoking policy + signage | Documentation only |
| **A03 Ventilation Design** | Precondition | Adequate ventilation; Option 4 = CO2 ≤900 ppm (continuous) | Yes — CO2 |
| **A04 Construction Pollution Management** | Precondition | IAQ management plan during construction | Documentation only |
| **A05 Enhanced Air Quality** | Optimization | Enhanced PM + formaldehyde + inorganic gas thresholds | Partial (PM yes; HCHO needs Plus/WELL; speciated VOCs no) |
| **A06 Enhanced Ventilation Design** | Optimization | Increased outdoor air; Option 4 = CO2 monitoring | Yes — CO2 |
| **A07 Operable Windows** | Optimization | Operable windows in regularly occupied spaces | Documentation / physical |
| **A08 Air Quality Monitoring & Awareness** | Optimization | ≥3 parameters monitored + display; monitor deployment | Yes — MICA qualifies for monitor deployment |

### A01 Parts detail

| Part | Parameter | Verification options |
|------|-----------|---------------------|
| Part 1 | PM2.5, PM10 | Performance test OR continuous sensor data (≥1 month) |
| Part 2 | Organic gases (TVOC / formaldehyde) | Lab test OR continuous TVOC monitoring (Option 2) |
| Part 3 | Inorganic gases (CO, O3, NO2) | Performance test OR continuous sensor data |
| Part 4 | Radon | Radon test — **MICA does not measure this** |
| Part 5 | Broader parameter set | Periodic measurement (informative, annual On-going Data Report) |

### A03 CO2 threshold (continuous monitoring)
Per WELL Addenda: Option 4 (ventilation monitoring) uses **CO2 ≤900 ppm** for continuous sensor-based verification.
Note: CLAUDE.md quick-reference shows ≤800 ppm — that is the standard WELL A03 operational target; 900 ppm is the continuous monitoring option threshold.

## Thermal Comfort Concept: Features T01–T07

| Feature | Type | What it requires | Sensor-verifiable with MICA |
|---------|------|-----------------|----------------------------|
| **T01 Thermal Performance** | Precondition | ASHRAE 55 comfort; Option 2 = long-term thermal sensor data | Yes — temperature + RH |
| **T02 Verified Thermal Comfort** | Optimization | ≥80% occupant satisfaction survey | Survey only |
| **T03 Thermal Zoning** | Optimization | Individual or zone control | Documentation / physical |
| **T04 Individual Thermal Control** | Optimization | Personal control options | Documentation / physical |
| **T05 Radiant Thermal Comfort** | Optimization | Mean radiant temperature assessment | Requires MRT sensor |
| **T06 Thermal Comfort Monitoring** | Optimization | Monitor deployment (temp + RH); informative | Yes — monitor deployment, no threshold data needed |
| **T07 Humidity Control** | Precondition (under Innovation Option 3) / Optimization | RH 30–60% during ≥98% of operating hours; Option 3 = long-term sensor data | Yes — RH |

### T07 detail
- Target: RH 30–60%
- Verification via long-term sensor data (Option 3): submit ≥1 month of RH data showing 98% of occupied hours within range

## WELL Certification Process Stages

1. **Gap analysis / feasibility** — assess against WELL feature thresholds; identify pre/opt gaps
2. **Documentation preparation** — LoAs (Owner, Contractor, MEP, Architecture), PIP template, scorecards, policies
3. **Construction / occupancy** — ≥50% occupancy for fit-out; ≥1 month after certificate of occupancy
4. **Performance Verification** — on-site testing by WELL Performance Testing Agent OR continuous sensor data (≥1 calendar month)
5. **Submission and review** — preliminary review 4–6 weeks; concurrent review 20–25 business days
6. **Certification awarded** — IWBI grants level based on points
7. **Annual renewal** — ongoing monitoring, maintenance reports
8. **Full recertification** — every 3 years

### Common failure modes
- Missing or unsigned LoAs / WELL Agreement
- Scope mismatch (enrolled type vs. actual project)
- Incomplete PIP or inaccurate dashboard info
- Missing A01 Part 4 radon test
- Sensor calibration not documented
- Insufficient data period (< 1 calendar month for performance-based)
- Failing any precondition (A01–A04 in Air; T01, T07 in Thermal)
